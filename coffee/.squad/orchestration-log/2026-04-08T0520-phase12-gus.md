## Key Decisions
✓ Tailwind via Vite plugin (simpler, faster)  
✓ Theme tokens decoupled from Tailwind (CSS vars in `:root`/`[data-theme="dark"]`)  
✓ `useTheme` owns theme state (single source of truth)  
✓ `color-mix()` for navbar translucency (modern CSS, no extra layers)  
✓ `SectionWrapper` as presentation boundary (consistent UX)  
✓ No App.css boilerplate (all globals in theme.css)
